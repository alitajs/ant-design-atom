## API

| 参数        | 说明                           | 类型   | 默认值                  | 是否必填 |
| ----------- | ------------------------------ | ------ | ----------------------- | -------- |
| color       | 背景色                         | string | `rgba(24, 144, 255, 1)` | 否       |
| percent     | 实际百分比                     | number |                         | 是       |
| target      | 预期百分比                     | number | 100                     | 否       |
| strokeWidth | 进度条高度                     | number | 8                       | 否       |
| targetLabel | 鼠标覆盖预期百分比后的文字提示 | string | ''                      | 否       |
