## API

| 参数    | 说明               | 类型               | 默认值 | 是否必填 |
| ------- | ------------------ | ------------------ | ------ | -------- |
| title   | 标题               | `string` or `node` | ''     | 是       |
| total   | 总数               | `string` or `node` | ''     | 否       |
| height  | number             | number             | 1      | 否       |
| action  |                    | node               |        | 否       |
| footer  |                    | node               |        | 否       |
| avatar  | 头像               | node               | ''     | 否       |
| loading | 是否在加载中的标识 | boolean            | false  | 否       |
